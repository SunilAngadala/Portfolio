## Project: LogLens

?? **Live:** https://runloglens.vercel.app/

**LogLens** is a free, open source side project I built out of a real pain point. Every time I hit a production incident, I was either paying for Splunk/Datadog or manually grepping through thousands of lines of logs. I wanted something instant, private, and free � so I built it.

### What it does

LogLens is a client-side log analysis tool built with **React** and **Vite**. Paste or upload any log file and it:

- Groups repeated error patterns automatically
- Detects **17 log levels** (ERROR, WARN, INFO, DEBUG, FATAL, TRACE, and more)
- Identifies HTTP status codes and highlights anomalies
- Suggests possible fixes based on detected patterns
- Exports shareable analysis reports

### AI-Powered Analysis

LogLens integrates with the three leading AI providers — bring your own API key and get deep, contextual analysis of your logs directly in the browser:

| Provider | Model |
|---|---|
| **OpenAI** | GPT-4o / GPT-4 Turbo |
| **Google Gemini** | Gemini 1.5 Pro |
| **Anthropic Claude** | Claude 3.5 Sonnet |

Your API key is used directly from the browser and is never stored or sent to any intermediary server. Switch between providers at any time to compare analysis quality.

### Why it's different

**Zero dependencies. Zero data leaving your browser.** Everything runs entirely on the client — no server, no backend, no tracking, no cost. Your logs and API keys never leave your machine.
